Matteo Capucci (Nov 25 2020 at 12:54):

I want to point out that using III is not necessary to express naturality. Well, it is if you do it the way Kelly does, using the unitors of V\mathcal VV, but it is not if you just write down the equation expressing naturality.
What I mean here is that given two V\mathcal VV-functors F,G:C→DF, G : \mathbf C \to \mathbf DF,G:C→D, one can define a V\mathcal VV-natural transformation α:F→G\alpha:F \to Gα:F→G to be an obCob \mathbf CobC-indexed collection of morphisms αX:P→D(FX,GX)\alpha_X : P \to \mathbf D(FX, GX)αX​:P→D(FX,GX).
Then you ask naturality in this way: given f:Q→C(X,Y)f : Q \to \mathbf C(X,Y)f:Q→C(X,Y), you want

αX⊗(f;GX,Y);∘D=((f;FX,Y⊗αY);∘D\alpha_X \otimes (f;G_{X,Y}) ; \circ_D = ((f;F_{X,Y} \otimes \alpha_Y) ; \circ_DαX​⊗(f;GX,Y​);∘D​=((f;FX,Y​⊗αY​);∘D​

Matteo Capucci (Nov 25 2020 at 13:04):

Fixing PPP for all the αX\alpha_XαX​ seems unavoidable, since there would be no way to express naturality then (basically, the left and right hand of the above equation would have a different source!)
To relax this, one could go galaxy brain and use the end definition of natural transformations:
[F,G]=∫X∈CD(FX,GX)[F,G] = \int_{X \in \mathbf C} \mathbf D(FX,GX)[F,G]=∫X∈C​D(FX,GX)
Then one would add a weight W:C→VW : \mathcal C \to \mathcal VW:C→V and define WWW-weighted natural transformations to be
[F,G]W=∫X∈C[WX,D(FX,GX)][F,G]^W = \int_{X \in \mathbf C} [WX, \mathbf D(FX,GX)][F,G]W=∫X∈C​[WX,D(FX,GX)]
