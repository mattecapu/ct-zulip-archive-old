Morgan Rogers (he/him) (Jun 30 2023 at 14:01):

John Baez said:

Okay, good. I like to think semantically, i.e. describing theories by saying what their models are like. So if I'm not confused, I would say Set/X\mathsf{Set}/XSet/X is "the classifying topos for the theory of an element of XXX", meaning that to specify a geometric morphism from some topos to Set/X\mathsf{Set}/XSet/X we just choose an element of XXX.

Unfortunately this is not the case, although it's true that the models in Set\mathsf{Set}Set look like that!
As @Max New pointed out earlier, this classifies the theory of flat functors on XXX, where XXX is viewed as a discrete category. What is a [[flat functor]]? Unfortunately, the nLab page is insufficiently detailed about this, but I can elaborate for you the definition of topos-valued flatness via the definition of site-valued flatness. :sweat_smile:

In this particular case, we require that ∐x∈XF(x)\coprod_{x \in X} F(x)∐x∈X​F(x) covers the terminal object, which is what it means for the image to be inhabited. We require that for any span F(x1)←Y→F(x2)F(x_1) \leftarrow Y \rightarrow F(x_2)F(x1​)←Y→F(x2​) there exists a covering family over YYY by objects of the form F(x)F(x)F(x) such that there are spans x1←x→x2x_1 \leftarrow x \rightarrow x_2x1​←x→x2​ whose image under FFF factor the original span. When x1≠x2x_1 \neq x_2x1​=x2​ there are no such spans(!) so we require that the empty sieve covers YYY, which is to say that YYY is initial. In other words, F(x1),F(x2)F(x_1),F(x_2)F(x1​),F(x2​) are disjoint. Similarly, taking x1=x2x_1 = x_2x1​=x2​, we conclude that the projections from F(x1)×F(x1)F(x_1) \times F(x_1)F(x1​)×F(x1​) must be equal, which (by a little diagram chase) means that F(x1)F(x_1)F(x1​) must be a subterminal object (i.e. an object whose unique morphism to the terminal object is a monomorphism). There is a third condition involving parallel arrows in XXX but there are no interesting such pairs to worry about.

In summary, Set/X\mathsf{Set}/XSet/X classifies the theory of XXX-indexed disjoint partitions of the terminal object.

There are other ways to figure this out / check it. A functor out of Set/X\mathsf{Set}/XSet/X which preserves colimits is determined by the representables y(x)\mathbf{y}(x)y(x) that @finegeometer identified, we only need to check which ones preserve finite limits. The representables are subterminal and disjoint, and both of these properties are preserved when finite limits are, which brings us pretty rapidly to the conclusion above.
