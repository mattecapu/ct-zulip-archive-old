I saw a related idea in "Seven Sketches in Compositionality", page 13. A preorder can be thought of as a category, where you have a single morphism from aaa to bbb exactly when a≤ba \leq ba≤b. So the idea is to switch from a preorder category to an equivalent(?) partial order category where there are no two distinct objects that are isomorphic.

Consider a two-element set {a,b}\{a,b\}{a,b} equipped with the discrete topology {∅,{a},{b},{a,b}}\{∅, \{a\},\{b\}, \{a,b\}\}{∅,{a},{b},{a,b}}. Here are two distinct open covers: T={{a},{a,b}}T=\{\{a\}, \{a,b\}\}T={{a},{a,b}} and S={{b},{a,b}}S=\{\{b\},\{a,b\}\}S={{b},{a,b}}. Each set of TTT is contained in a set of SSS, and each set of SSS is contained in a set of TTT. I think that means that TTT and SSS are refinements of one another, while being distinct. So, it seems like the open covers of a finite topological space don't always form a partial order under refinement.

That being said, it does feel a little bit like cheating to include {a,b}\{a,b\}{a,b} as a set in open covers of {a,b}\{a,b\}{a,b}. Maybe the answer would change if we didn't allow this sort of thing?

David Egolf (Sep 05 2022 at 17:35):

Actually, if we have the set {a,b,c}\{a,b,c\}{a,b,c} with the discrete topology, we have open covers with subsets {{a},{b,c}}\{\{a\}, \{b,c\}\}{{a},{b,c}} and {{a},{b},{c},{b,c}}\{\{a\}, \{b\}, \{c\}, \{b,c\}\}{{a},{b},{c},{b,c}}. Each subset in the first open cover is contained in a subset in the second open cover, and each subset in the second open cover is contained in a subset in the first open cover. So, two open covers can be refinements of one another for a finite topological space even when neither cover contains the whole set (in this case {a,b,c}\{a,b,c\}{a,b,c}) as a subset.

David Egolf (Sep 05 2022 at 17:43):

A question that occurs to me: if one has an open cover MMM and one closes it downwards "(so that whenever U is in the cover and V is contained in U then V is in the cover)", is the resulting open cover D(M)D(M)D(M) a refinement of MMM? And is MMM a refinement of D(M)D(M)D(M)? That is, is D(M)D(M)D(M) isomorphic to MMM in the preorder of open covers for some topological space under refinement?
Let's see. Each subset of MMM is contained in D(M)D(M)D(M), because D(M)D(M)D(M) contains every subset of MMM (plus possible some additional ones). Each subset of D(M)D(M)D(M) was either in MMM, or it was a subset of one in MMM (by definition of D(M)D(M)D(M)). So each subset of D(M)D(M)D(M) is contained in a subset of MMM. It seems that D(M)D(M)D(M) and MMM are refinements of one another, if I did this right.

David Egolf (Sep 05 2022 at 17:48):

I think that means that closing open covers downwards and working with those ("covering sieves"?) corresponds to picking a skeleton of the preorder of open covers of a topological space under refinement: we pick one open cover from each isomorphism class to work with.
