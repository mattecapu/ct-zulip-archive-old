Topic: Name for the inverse of delooping

{% raw %}

Thibaut Benjamin (May 29 2020 at 15:02):

It is well known that monoidal categories are equivalent 222-categories with one object, and the functor sending a monoidal category onto a category with one object is usually called the delooping. Now as this is an equivalence it has an inverse which sends any one object 222-category onto its corresponding monoidal category. Is there widely accepted name for this inverse functor?
