Ben Logsdon (Aug 31 2023 at 19:58):

I believe the original paper behind Jade Master's answer is "Algebraic Real Analysis", by Peter Freyd:

http://www.tac.mta.ca/tac/volumes/20/10/20-10.pdf

I'll summarize the ideas here since I like them, but everything below can be found in that paper.

Broadly speaking, one can think of an algebra as an object equipped with operations that combine (potentially multiple) elements to form a single new element, while one can think of a coalgebra as an object equipped with operations that can unpack an element to form (potentially multiple) elements. One of the simplest examples of a coalgebra are comonoids in the category of sets given by duplication and deletion. Given a set AAA and an element a∈Aa \in Aa∈A, we can "unpack" aaa to form an element of A×AA \times AA×A by simply copying it: (a,a)(a,a)(a,a), thus unpacking aaa into two elements of AAA. Likewise, we can "unpack" aaa into an element of the terminal set 1\mathbf{1}1 simply by forgetting about aaa entirely, thus unpacking aaa into zero elements of AAA.

The unit interval is a coalgebra in the following sense. It is a set with two distinguished points, ⊥\bot⊥ (the number 000) and ⊤\top⊤ (the number 111). It supports a "split" operation which splits the unit interval into two subintervals, one subinterval from 000 to 12\frac1221​, and one subinterval from 12\frac1221​ to 111. This splitting operation unpacks a single point in the interval into either a point in the left subinterval or a point in the right subinterval. (If we're plugging in 12\frac1221​, things get more tricky, as it is a member of both subintervals.) Each subinterval is isomorphic to the unit interval itself, and we can continue the splitting process ad infinitum.

In more detail (see the nLab page or the fourth page of Freyd's paper), let Set∗∗\textnormal{Set}_{**}Set∗∗​ be the category whose objects are sets with two distinguished elements (i.e. bipointed sets), ⊥\bot⊥ and ⊤\top⊤ and whose morphisms are functions which preserve these distinguished elements. If XXX and YYY are bipointed sets, then we can define their ordered wedge X∨YX \vee YX∨Y by taking the disjoint union of the two sets and identifying the top of XXX with the bottom of YYY. Then the unit interval is a terminal coalgebra for the cofunctor X↦X∨XX \mapsto X \vee XX↦X∨X. A coalgebra for this endofunctor consists of a map X→X∨XX \to X \vee XX→X∨X, which is the splitting operation mentioned in the above paragraph.

Nathaniel Virgo (Sep 01 2023 at 06:44):

Rational numbers (even with denominators that are not powers of two) map to cyclic sequences, e.g.

  • 23\frac{2}{3}32​ is in the interval [12,1][\frac{1}{2},1][21​,1] and it maps back to 2⋅23−1=132\cdot\frac{2}{3} -1 = \frac{1}{3}2⋅32​−1=31​.

  • 13\frac{1}{3}31​ is in the interval [0,12][0,\frac{1}{2}][0,21​] and it maps back to 2⋅13=232\cdot \frac{1}{3} = \frac{2}{3}2⋅31​=32​.

so 2/32/32/3 maps to the real number (in binary) 0.1010101…0.1010101\dots0.1010101…. Irrational numbers have nonrepeating binary expansions, so the splitting process ends up not repeating.

Mike Shulman (Sep 01 2023 at 16:45):
