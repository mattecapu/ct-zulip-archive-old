No that is not the fact Morgan was referring to. There is a correspondence between elements of a set SSS and functions from the terminal set 1={∗}\mathbf{1} = \{*\}1={∗} to SSS. Here it is:

  • Given an element x∈Sx \in Sx∈S, you define the function fx:1→Sf_x: \mathbf{1} \to Sfx​:1→S that sends the unique element ∗∈1*\in \mathbf{1}∗∈1 to xxx.
  • Given a function f:1→Sf:\mathbf{1} \to Sf:1→S, you define the element xf=f(∗)∈Sx_f = f(*) \in Sxf​=f(∗)∈S.
  • You should check that these two processes are inverses, i.e., xfx=xx_{f_x} = xxfx​​=x and fxf=ff_{x_f} = ffxf​​=f. You can then conclude S≅Hom(1,S)S \cong \mathrm{Hom}(\mathbf{1},S)S≅Hom(1,S).

Unimportant aside, note that a common notation for the set of functions from XXX to YYY is YXY^XYX. What we proved above is that S≅S1S \cong S^{\mathbf{1}}S≅S1, which looks pretty neat imo.
