There should be a way to do it. Here is something simpler, maybe helpful, that doesn't use the monoid structure on DDD at all.

There's a morphism called internal composition:

∘x,y,z:(x⊸y)⊗(y⊸z)→(x⊸z)\circ_{x,y,z} :(x \multimap y) \otimes (y \multimap z) \rightarrow (x \multimap z)∘x,y,z​:(x⊸y)⊗(y⊸z)→(x⊸z)

in any symmetric monoidal closed category. (In fact a monoidal closed category is sufficient if we tensor things in the correct order, but I'm too lazy to remember the correct order so I'll use a symmetric monoidal closed category.) Using the hom-tensor adjunction we can get the above morphism if we have a morphism

x⊗(x⊸y)⊗(y⊸z)→z(⋆)x \otimes (x \multimap y) \otimes (y \multimap z) \rightarrow z \qquad (\star)x⊗(x⊸y)⊗(y⊸z)→z(⋆)

But notice that in any symmetric monoidal closed category there are evaluation morphisms

x⊗(x⊸y)→y x \otimes (x \multimap y) \to y x⊗(x⊸y)→y

y⊗(y⊸z)→z y \otimes (y \multimap z) \to z y⊗(y⊸z)→z

By suitably composing/tensoring these we get the desired morphism (⋆)(\star)(⋆).

Yes, this is the notation used in linear logic for the linear implication, ie. if x,yx,yx,y are proposition, x⊸yx \multimap yx⊸y is the proposition which means "from one unit of xxx, I get one unit of yyy" which is not the same that x⊗x⊸yx \otimes x \multimap yx⊗x⊸y which means "from two units of "x", I get one unit of yyy"

Jean-Baptiste Vienney (May 25 2023 at 15:19):

Categorically, the connectors ⊗\otimes⊗ and ⊸\multimap⊸ of linear logic correspond to symmetric monoidal closed categories.

Jean-Baptiste Vienney (May 25 2023 at 15:20):

Whereas, the connectors ∧\wedge∧ and ⇒\Rightarrow⇒ of intuitionist logic correspond to cartesian closed categories.

John Baez (May 25 2023 at 15:39):

David Egolf said:

I haven't seen the notation x⊸yx \multimap yx⊸y before. Is this another notation for [x,y][x, y][x,y] where [−,−][-,-][−,−] is an internal hom functor?

Yes, it's an internal hom. This notation emphasizes that we've got an internal hom related to a tensor product ⊗\otimes⊗ that's not necessarily cartesian. For example in Set or any other topos we've got

hom(X×Y,Z)≅hom(X,ZY) \mathrm{hom}(X \times Y, Z) \cong \mathrm{hom}(X, Z^Y) hom(X×Y,Z)≅hom(X,ZY)

but in Vect we've got

hom(X⊗Y,Z)≅hom(X,Y⊸Z) \mathrm{hom}(X \otimes Y, Z) \cong \mathrm{hom}(X, Y \multimap Z) hom(X⊗Y,Z)≅hom(X,Y⊸Z)

where Y⊸ZY \multimap ZY⊸Z is the vector space of linear maps from YYY to ZZZ. But as Jean-Baptiste noted, it's mainly linear logicians who use the symbol ⊸\multimap⊸. Also category theorists. [Y,Z][Y , Z][Y,Z] is perhaps more common.
