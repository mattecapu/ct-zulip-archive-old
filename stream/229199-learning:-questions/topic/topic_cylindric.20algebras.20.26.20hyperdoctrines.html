Reid Barton (Apr 05 2023 at 07:59):

[[hyperdoctrine]] [[Boolean hyperdoctrine]]
Seems like the closest comparison would be between a cylindric algebra and a Boolean hyperdoctrine on the opposite of the category of finite sets. The main difference is that in a cylindric algebra we combine all the formulas into a single Boolean algebra AAA, while in a hyperdoctrine we have a separate Boolean algebra P(n)P(n)P(n) of formulas in n free variables for each nnn.
Basically, it is similar to the difference between working with or without an explicit context.

Reid Barton (Apr 05 2023 at 09:13):

The idea would be: We could define one Boolean hyperdoctrine by picking our favorite language of first-order logic and then letting P(I)P(I)P(I) be the set of formulas in this language with free variables from III, say denoted by xix_ixi​, i∈Ii \in Ii∈I. (Then, we should quotient these formulas by bi-implication but I'll ignore this.) For fixed III, P(I)P(I)P(I) is a Boolean algebra because we can form conjunctions, disjunctions, negations of formulas.
Now if we have a map f:I→Jf : I \to Jf:I→J of sets, then we get a homomorphism P(I)→P(J)P(I) \to P(J)P(I)→P(J) of Boolean algebras that takes every occurrence of xix_ixi​, and replaces it with xf(i)x_{f(i)}xf(i)​. In other words, it carries out a substitution where we are only allowed to substitute variables by other variables.

Reid Barton (Apr 05 2023 at 09:18):

The map P(f):P(I)→P(J)P(f) : P(I) \to P(J)P(f):P(I)→P(J) is supposed to have a left adjoint, which turns out to encode the existential quantifier ∃xα\exists x_{\alpha}∃xα​ if fff is an inclusion I→I∪{α}I \to I \cup \{\alpha\}I→I∪{α}, and to encode imposing an equality xβ=xγx_\beta = x_\gammaxβ​=xγ​, if fff is the function that identifies two elements β\betaβ, γ\gammaγ of III. So, that corresponds somehow to the ccc's and ddd's of the cylindric algebra.

Reid Barton (Apr 05 2023 at 09:27):

I'm using "formulas" because this is somehow the original motivating example, but it's also important to note that the elements of the Boolean algebras P(I)P(I)P(I) (or of a cylindric algebra) can be anything, like subsets of 2I2^I2I; then we're working with the semantics of the formulas, and not the syntax itself.
