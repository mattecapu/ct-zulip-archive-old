In case anyone reading this wants an example: if E=[Cop,Set]E = [C^{op}, \textsf{Set}]E=[Cop,Set] is a presheaf topos and ccc is an object of CCC, then

E(C(−,c),−):E→SetE(C(-, c), -): E \to \textsf{Set}E(C(−,c),−):E→Set

preserves colimits and has a right adjoint, by the Yoneda lemma. It's worthwhile computing this example where the amazing right adjoint exists. (And this example could be beefed up to the internal case, if we assume for example that CCC has products.)
