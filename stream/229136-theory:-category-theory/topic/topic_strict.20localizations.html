Is there anything written about "strict" localizations, ie the universal solution to mapping a set of arrows in a category to identities?

Specifically I'm looking for references for the following two statements.

  1. 222-categorical universal property: given a small category C\mathbb CC and a set III of arrows in C\mathbb CC, there exists a functor E:C→L(C,I)E:\mathbb C\to L(\mathbb C, I)E:C→L(C,I) which which sends III-maps to identities, and such that for every category X\mathbb XX, precomposition with EEE induces an isomorphism of categories between [L(C,I),X][L(\mathbb C, I),\mathbb X][L(C,I),X] and the full subcategory of [C,X][\mathbb C,\mathbb X][C,X] on functors sending III-maps to identities.
  2. Given a diagram D:J→CatD:\mathbb J\to\mathbf{Cat}D:J→Cat of small categories, colim(D)\mathsf{colim}(D)colim(D) can be computed as strict localization of ∫D\int D∫D (the covariant Grothendieck construction) at the cocartesian arrows.

Thanks!
