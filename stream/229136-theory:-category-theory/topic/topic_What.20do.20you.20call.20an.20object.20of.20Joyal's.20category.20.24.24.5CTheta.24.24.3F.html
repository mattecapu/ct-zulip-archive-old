Tim Campion (Feb 22 2023 at 17:57):

We have:

  • An object of the simplex category Δ\DeltaΔ is called a simplex.
  • An object of a cube category □\square□ is called a cube.
  • An object of the globe category G\mathbb GG is called a globe (or sometimes a cell).
  • An object of Joyal's category Θ\ThetaΘ is called what?

Whatever the answer I want it to be applicable to presheaves too as follows (where either C=Set\mathcal C = \mathsf{Set}C=Set or else I'm talking about generalized elements):

  • An element x∈X([n])x \in X([n])x∈X([n]) is a simplex in XXX, where X:Δop→CX : \Delta^{op} \to \mathcal CX:Δop→C is a simplicial object and [n]∈Δ[n] \in \Delta[n]∈Δ.
  • An element x∈X(□n)x \in X(\square^n)x∈X(□n) is a cube in XXX, where X:□op→CX : \square^{op} \to \mathcal CX:□op→C is a cubical object and □n∈□\square^n \in \square□n∈□.
  • An element x∈X(Gn)x \in X(\mathbb G_n)x∈X(Gn​) is a globe in XXX, where X:Gop→CX : \mathbb G^{op} \to \mathcal CX:Gop→C is a globular object and Gn∈G\mathbb G_n \in \mathbb GGn​∈G.
  • An element x∈X(θ)x \in X(\theta)x∈X(θ) is a what? in XXX, where X:Θop→CX : \Theta^{op} \to \mathcal CX:Θop→C is a Θ\ThetaΘ-presheaf and θ∈Θ\theta \in \Thetaθ∈Θ.

I suppose that presheaves on Θ\ThetaΘ are often called "cellular sets", and so maybe the answer should be "cell". But this is a bit suboptimal because it clashes with the use of "cell" to mean "globe", and also with the more generic use of "cell" in the setting of an arbitrary presheaf category.

Θop\Theta^{op}Θop is a category of disk bundles, but even ignoring the contravariance, I'm not sure "disk bundle" would work in the second set of bullet points. Probably whatever the term, it should be one word.
