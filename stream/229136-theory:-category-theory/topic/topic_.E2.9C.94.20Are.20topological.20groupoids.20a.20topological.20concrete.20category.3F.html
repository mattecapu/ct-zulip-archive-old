Chris Grossack (they/them) (Aug 19 2023 at 18:37):

Yes!

More generally, fix your favorite essentially algebraic theory T\mathbb{T}T. Then Mod(T,Top)\mathsf{Mod}(\mathbb{T},\mathsf{Top})Mod(T,Top), the models of T\mathbb{T}T in Top\mathsf{Top}Top, will be topological over Mod(T,Set)\mathsf{Mod}(\mathbb{T}, \mathsf{Set})Mod(T,Set) via the obvious forgetful functor.

Embarrassingly, I don't actually have a reference for this (though it may be easy to prove? I've never tried.), but it's stated as fact in Section 2.2 of Topological Functors as Total Categories

Relatedly, if anyone happens to have a reference that actually proves this, I would be super interested in it. It's almost certainly "do the obvious thing", but I don't have time to work out the details right now. For context, the "obvious thing" is this:

Given a lex functor F:T→SetF : \mathbb{T} \to \mathsf{Set}F:T→Set, a family of functors Gα:T→TopG_\alpha : \mathbb{T} \to \mathsf{Top}Gα​:T→Top, and a bunch of natural transformations F⇒UGαF \Rightarrow UG_\alphaF⇒UGα​, we want to find a "final lift" F~:T→Top\tilde{F} : \mathbb{T} \to \mathsf{Top}F~:T→Top so that UF~=FU\tilde{F} = FUF~=F. We should be able to take the lift "pointwise" in the sense that, at each object t∈Tt \in \mathbb{T}t∈T, we get a source Ft→UGαtFt \to UG_\alpha tFt→UGα​t, which we can lift to an object Ft~\widetilde{Ft}Ft. It's not hard to check that this is compatible with arrows in T\mathbb{T}T as well, so that we can define a functor F~(t)=Ft~\tilde{F}(t) = \widetilde{Ft}F~(t)=Ft. This is almost certainly the right functor.

But officially we need to check that it's an initial lift of the natural transformations (this should be easy) and moreover that it still preserves finite limits (this seems potentially annoying, but I haven't tried)
