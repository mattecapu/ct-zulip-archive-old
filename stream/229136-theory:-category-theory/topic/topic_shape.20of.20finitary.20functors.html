James Deikun (May 12 2022 at 11:27):

When a category CCC has finite limits, then a functor F:C→DF : C \to DF:C→D preserves these limits exactly when each comma category (d/F)(d/F)(d/F) is cofiltered.

Is there a similar statement for finitary functors from categories with filtered colimits, relating the colimit-preservation of the functor to the shape of its comma categories (d/F)(d/F)(d/F) or perhaps (F/d)(F/d)(F/d)?

For a functor F:C→EF : C \to EF:C→E, the coYoneda extension RanFop:[C,Set]op→[E,Set]opRan_F^{\mathrm{op}} : [C,\bold{Set}]^{\mathrm{op}} \to [E,\bold{Set}]^{\mathrm{op}}RanFop​:[C,Set]op→[E,Set]op is computed by the limit

(RanF X)(c)≃lim((c/F)→C→XSet) ,(\mathrm{Ran}_F\,X)(c) \simeq \mathrm{lim}\left((c/F) \to C \stackrel{X}{\to} \bold{Set}\right)\,,(RanF​X)(c)≃lim((c/F)→C→XSet),

so this should preserve filtered colimits exactly when the comma categories (c/F)(c/F)(c/F) are L-finite. Because the coYoneda embedding ʎ:C→[C,Set]opʎ : C \to [C,\bold{Set}]^{\mathrm{op}}ʎ:C→[C,Set]op preserves colimits whenever they exist, then for categories C,EC, EC,E with all filtered colimits, these should be preserved by FFF exactly when all comma categories (c/F)(c/F)(c/F) are L-finite.

It's interesting that this also involves the "coslice-like" categories (c/F)(c/F)(c/F) rather than the "slice-like" (F/c)(F/c)(F/c) because of the changes in variance cancelling each other out...
