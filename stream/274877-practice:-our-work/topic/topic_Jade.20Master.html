Jade Master (Jul 28 2023 at 15:12):

Yes,

Spencer Breiner said:

In the fibrational decompositions, you are allowed to have nonzero edges in the base graph, but only the zero-length edges lift. Is that correct?

Yes, except the nonzero edges must have weight ∞\infty∞. This is because the base must be the free [0,∞][0,\infty][0,∞]-graph on B-graph. I never said this: but [0,∞][0,\infty][0,∞] is equipped with the reverse of the usual ordering, so the condition necessary to be an R-graph morphism is that H(x,y)≥G[0,∞](f(x),f(y))H(x,y) \geq G_{[0,\infty]} (f(x),f(y))H(x,y)≥G[0,∞]​(f(x),f(y)) which is always true if the latter weight is 0 and never true if the latter weight is ∞\infty∞. Therefore, only the 0-weigted edges lift.
