Hi everybody, I'm a french student in last year of engineering school and second year of master degree in mathematics/theoretical computer science. I have some ideas related to linear logic which I want to develop. Let me explain this. At the beginning, there was linear logic (1987). After that came differential linear logic (around 2002) and differential categories (2006). There is also "A Categorical Quantum Logic" (2005) which is basically compact multiplicative linear logic with bi-products. I'm interested by the fact that some mathematical mechanism can be described in a more combinatorial way by logical systems, forgetting the details of the model and concentrating on the pure calculus. In linear logic, we are dealing with kind of linear mechanisms. In differential linear logic, we are dealing with linearity and differentiation. If we want to interpret this in vector spaces, we need infinite dimensional ones. What has been done with "A Categorical Quantum Logic", is adapting linear logic to the category of finite dimensional vector spaces (or finite sets if you want) but without the exponentials. Recently, @JS Pacaud Lemay (he/him) proved that the category of finite dimensional vector spaces cannot be endowed with a nontrivial structure of co-differential category. Differential or co-differential categories are similar to categorical models of differential linear logic where you forget additional structure and generalize a little to concentrate on differentiation. The intuition I think is that the exponentials of linear or differential linear logic are not compatible with finite dimensionality, they always produce infinite dimensional vector spaces. This is where a fifth system come in the game : bounded linear logic (1991), more precisely a very simple version of bounded linear logic. In the introduction of the paper, Girard (I guess because of the style but there are other writers) explains clearly that the goal of this logic is to concentrate on finite stuff. To do this he uses indexed exponentials which are in the simple version !n!_{n}!n​ where n≥0n \ge 0n≥0 give the maximum number of times you can use the resource. I made the link with the work of JS, and it gives me the idea to introduce a new kind of differential category with such an indexed exponential, which I name graded differential categories. We are actually discussing of that with JS. Well, this is an interesting beginning, it gives some categorical structure to talk about polynomials for instance. But, there is more exciting. I want to see how to extract more dynamics from the category of finite dimensional vector spaces and doing a logic with that. One solution is to add such indexed exponentials to the "Categorical Quantum Logic". So, in the logic, you will have !nA!_{n}A!n​A and this is auto-dual : (!nA)⊥=!nA⊥(!_{n}A)^{\perp} = !_{n}A^{\perp}(!n​A)⊥=!n​A⊥. The most natural way to interpret this logic is to say that AAA is a finite dimensional vector space and that !nA=A⊗sn!_{n}A = A^{\otimes_{s}n}!n​A=A⊗s​n. So this give you a logic of finite dimensional vector spaces and symmetric powers. I think a cool name could be "Linear Logic S" with a S like symmetric powers but also because it seems very related to λ\lambdaλ-rings, but expressed with the operators sns^{n}sn and not the λn\lambda^{n}λn. I think this logic is really really interesting : it should express the equations of a λ\lambdaλ-ring but in asymmetrical manner : you have all the natural morphisms and if you take the isomorphism classes of propositions, you get a λ\lambdaλ-ring. The cut elimination seems really combinatorial and more beautiful than the one of linear logic or differential linear logic. I'm really wondering if it can mean something in quantum mechanics. I think you can also interpret !nA!_{n}A!n​A by a truncated Fock space ⨁0≤k≤nA⊗sk\underset{0 \le k \le n}{\bigoplus}A^{\otimes_{s}k}0≤k≤n⨁​A⊗s​k, so it may signifies something. You also naturally have differentiation in this logic, but this is more differentiation of polynomials. This can really be an important logic for me : this is a descendant of all the descendants of linear logic, this can be related to quantum mechanic and this is similar to λ\lambdaλ-rings.

  • I'm interested by finitism. In linear logic, a proof of !A⊢B!A \vdash B!A⊢B can be interpreted as an algorithm which produces an output of type BBB from an input of type AAA, used as many times as you want (maybe an infinite number of time). If you use !n!_{n}!n​, a proof of !nA⊢B!_{n}A \vdash B!n​A⊢B is an algorithm which produce the output of type BBB by using at most nnn times the input of type A (or by using exactly nnn times the input, I'm not really sure about this, the original interpretation in bounded linear logic is the first but it can be coherent to interpret with the second). This seems really more realistic to use !n!_{n}!n​ than !!! to describe algorithms. In any case, it also express a symmetry in the input : if you say that you use nnn different entries of type A, which can be also an interpretation, the algorithm must be symmetric in these entries. For the finite dimensional normed vector spaces, the point is that when you consider the category where the morphisms are contractive linear maps (maps that don't increase the norm) instead of bounded maps, you get a *-autonomous category where the par and the tensor are different while for vector spaces they are both interpreted by the tensor product. Similarly, the plus and with are distinct while in vector spaces they are given by the direct sum. I would be interested in looking at exponentials and differentiation in this context but I haven't yet.
